# Task 1: Excel Data Analysis

## Project Overview

This project focuses on analyzing a sales dataset using Microsoft Excel to identify patterns, trends, distributions, and business insights.

The objective was to transform raw sales data into meaningful information that can support business decision-making.

---

## Objectives

* Calculate descriptive statistics
* Analyze data distributions
* Identify trends and outliers
* Generate business insights
* Build an analytical dashboard

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Conditional Formatting
* Statistical Functions

---

## Dataset Information

The dataset contains:

* 1,200 sales transactions
* Product information
* Customer details
* Payment methods
* Order statuses
* Revenue data

---

## Analysis Performed

### Descriptive Statistics

The following metrics were calculated:

* Mean
* Median
* Count
* Minimum Value
* Maximum Value

### Trend Analysis

The analysis examined:

* Product performance
* Revenue trends
* Payment preferences
* Order status distributions

### Outlier Detection

Outliers were identified using statistical techniques to detect unusually large transactions.

---

## Key Insights

### Sales Performance

* Chairs and Printers generated the highest revenue.
* Electronics and furniture products contributed significantly to overall sales.

### Customer Behavior

* Customers purchased an average of 3 items per order.
* Online payments were the most preferred payment method.

### Operational Insights

* Cancelled and Returned orders were relatively high.
* Opportunities exist to improve customer satisfaction and fulfillment processes.

### Revenue Analysis

* Average order value was approximately 1,054.
* Several high-value transactions were identified.
